ParkJam

Don't waste time looking for a parking space, let ParkJam find it for you!

The best way to find parking is to know beforehand where the spaces are. With ParkJam, we can let each other know about the available car parks.

ParkJam is a mobile app for Android™ that uses openly available geographic data and crowdsources parking availability information, so that you can conveniently find parking when coming to work or driving into town.

The application builds on linked data, and it also openly publishes the crowdsourced parking availability data. Further, it integrates additional related data sources, such as events and services, to provide rich value-adding features that will make you want to find parking through ParkJam and no other app!
