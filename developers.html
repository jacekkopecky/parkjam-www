for drivers…for car park operators…for developers

The best way to find parking is to know beforehand where the spaces are. ParkJam makes its data available for open reuse.

ParkJam is a mobile app for Android™ that uses openly available geographic data and crowdsources parking availability information, so that drivers can conveniently find parking when coming to work or driving into town.

The application builds on linked data, and it also openly publishes the crowdsourced parking availability data. We plan to make it easy to integrate third-party electronic systems so that we can have authoritative availability data, where available.

Initially, ParkJam expects to use the following data sources:

  • Linked Geo Data (data from Open Street Map)
  • SFPark (planned)
